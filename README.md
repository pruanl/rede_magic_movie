####Requisitos para iniciar o projeto:

Composer

PHP 7.3

Banco de dados (Postgres ou MySQL)

####Comandos para iniciar o projeto

composer install

copy .env.example .env

php artisan key:generate

(Preencher .env com informações do banco de dados)

php artisan migrate

php artisan db:seed

php artisan serve
